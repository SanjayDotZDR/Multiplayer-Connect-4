# Multiplayer Connect 4 (Java)

A multiplayer Connect 4 game implemented in Java using socket programming and multithreading. The project supports both terminal-based clients (via `nc`) and a Java GUI client, allowing multiple games to run concurrently through a threaded server architecture.
